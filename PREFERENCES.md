# Préférences Projet Jules Adventure

## Règles absolues — à respecter à chaque session

1. **Push automatique** : après CHAQUE modification, pousser sur GitHub → Netlify se met à jour automatiquement.

2. **Icônes Minecraft** : toujours utiliser les VRAIES icônes Minecraft (sprites base64 de la version Tablette). Jamais d'emojis pour remplacer des icônes MC.

3. **Demander avant de modifier** : toujours proposer et faire valider avant d'appliquer un changement.

4. **Images réelles** : utiliser de vraies images (online si nécessaire) — pas de générateurs CSS/canvas pour les fonds.

5. **UI full screen style jeu vidéo** : les menus doivent utiliser tout l'espace écran, ambiance AAA game.
